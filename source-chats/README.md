# source-chats

Здесь хранятся исходные технические чаты — сырой материал, из которого затем рождаются главы книги.

Это не главы. Это фрагменты реальных разговоров о программировании: вопросы, ответы, обсуждения, иногда с ошибками и уточнениями. Их нужно сохранять как есть, без литературной обработки.

## Как использовать

1. Положи новый чат сюда отдельным файлом, например `oop-vs-fp.md` или `js-errors.md`.
2. Возьми промт из [`../prompts/chapter-generation-prompt.md`](../prompts/chapter-generation-prompt.md).
3. Передай промт + содержимое чата модели — на выходе получится глава в стиле книги.
4. Готовую главу положи в корень репозитория рядом с остальными.

## Что важно

- Не редактируй чаты задним числом, чтобы «сгладить» их. Ошибки и заблуждения — часть материала: они становятся живыми моментами обучения в главе.
- Один чат — один файл. Если тема большая, лучше разбить её на несколько чатов и, соответственно, на несколько глав.
